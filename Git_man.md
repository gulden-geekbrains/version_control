# Команды Git

Создание репо

```sh
git init
````
добавить содержимое рабочей директории в индекс
```sh
git add
````

Создать коммит с коментом
```sh
git commit -m "комент"
````

Список коммитов в репо
```sh
git log
````

Список коммитов в репо (краткий)
```sh
git log --oneline
````

Переключение ветки
```sh
git checkout<имя_ветки>
````

Показать состояния файлов в рабочей директории и индексе

```sh
git status
`````` 

Вычисление разницы между любыми двумя коммитами 

```sh
git diff
````


Отображение всех веток

```sh
git branch
````

Добавить новую ветку

```sh
git branch <имя_ветки>
````

Удалить ветку

```sh
git branch -d <имя_ветки>
````
Залить данные с удаленного на локальный репо

```sh
git pull 
````
Залить данные с локального на удаленный репо

```sh
git push 
````
Клонировать репо

```sh
git clone  <адрес_удаленного_репо>
````
