# Инструкция для работы с Markdown

## Выделение текста
чтобы выделить текст курсивом необходимо обрамить его звёздочками (\*) например *вот так*
для полужирного текста необходимы двойные звёздочки (\*) например **вот так**

Также можно делать выделенный текст двумя нижними подчёркиваниями (__)
Например __вот так__
Для курсивного текста можно использовать одно подчёркивание (_)
Например _вот так_
Это нужно для того, чтобы _текст мог быть **полужирным** и курсивом одновременно_
