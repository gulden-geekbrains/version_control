## Команды для работы с GitHub
> "git clone" - клонирует чужой репозиторий с github в локальный репозиторий, "git clone URL (название папки)
> "git pull" - выкачивает данные с удаленного репозитория и делает слияние с локальным репозиторием
> "git push" - переносит изменения в нашем локальном репозитории на github в нашем аккаунте
