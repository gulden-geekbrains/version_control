## Работа с удаленными репозиториями

### git pull

git pull используется для извлечения и загрузки содержимого из удаленного репозитория и немедленного обновления локальногорепозитория этим содержимым.

### git push

git push позволяет передать изменения из локального репозитория в удаленный

### pull request

это запрос на интеграцию изменений из одной ветки в другую.
