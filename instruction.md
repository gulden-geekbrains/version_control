# Инструкция по языку MarkDown

## Стилизация текста

Обычный текст набираем как есть.

Новая строка.

*Курсив*

**Полужирный текст**

## Цитирование

> Первый уровень цитирования
>> Второгой уровень цитирования
>>> Третий уровень цитирования

## Списки

### Ненумированный список
* Лист1
* Лист2

### Нумированный список
1. Лист
2. Лист

### Вложенные списки

1. Пункт первый
    - Подпункт первый
    - Подпункт второй
2. Пункт второй
