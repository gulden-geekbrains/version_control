```sh

```

## Выделение текста

Для выделения полужирным ,необходимо обрамить текст двойной звездочкой,вот так: **Полужирный текст**

Для выделения текста курсивом ,необходимо обрамить его звёздочками, вот ток: *Курсив текст*
