# Работа с GitHub
команда "Git push" - добавляет изменения на сайт Git Hub (если работаем с файлом на компьютере в терминале)
команда "Git pull" - добавляет измения на компьютер (если работали на сайте Git Hub)
команда "git pull -- rebase" - вводим ее при конфликтах
