# Инструкция по языку MarkDown

## Стилизация текста

Обычный текст набираем как есть.

Новая строка.

Сохранить изменений
```sh
git add <filename> 
```

Добавить комментарий к сохранению
```sh
git commit -m "<внесенные изменения>"
```

Просмотр статуса файла
```sh
git status
```
