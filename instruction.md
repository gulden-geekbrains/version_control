## Команда позволяет перемещаться между ветками
``````
git checkout<Имя файла>
``````
##  команда позволяет слить ветки
``````
git merge <имя ветки>
``````
# Удалить уже слитую ветку

``````
git branch -d <имя ветки>
``````

# Увидеть лог коммитов с визуализацией между ними
``````
git log --graph
``````
