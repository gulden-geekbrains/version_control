## Работа с Githab

Клонирование внешнего репозитория на локальный ПК:

``sh
git clone <url-адрес репозитория>
``

Получение изменений и слияние с локалькой версией: 

``sh
git pull
``

Отправляет локальлную верчию репозитория на Githab:

``sh
git push
``
