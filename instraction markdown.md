# Команды по Markdown
## Заголовки
```sh
# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня
#### Заголовок четвёртого уровня
##### Заголовок пятого уровня
###### Заголовок шестого уровня
```
Пример

# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня
#### Заголовок четвёртого уровня
##### Заголовок пятого уровня
###### Заголовок шестого уровня


# Выделение текста
```sh
*курсив*  
_курсив_

**жирный**  
__жирный__

***жирный курсив***  
___жирный курсив___

~~зачеркнутый~~
```
Пример:

*курсив*  
_курсив_

**жирный**  
__жирный__

***жирный курсив***  
___жирный курсив___

~~зачеркнутый~~

# Списки
### Нумерованный список
```sh
1. Пункт первый
2. Пункт второй
3. Пункт третий
```
Пример:
1. Пункт первый
2. Пункт второй
3. Пункт третий

### Не нумерованный список
```sh
- Пункт первый
- Пункт второй
- Пункт третий
```
Пример:
- Пункт первый
- Пункт второй
- Пункт третий

### Ссылки
```sh
[Текст ссылки](https://www.example.com)
```
Притер:

[Текст ссылки](https://www.example.com)

### Цитаты
```sh
> Первый уровень цитирования
>> Второй уровень цитирования
>>> Третий уровень цитирования
```
Пример:

> Первый уровень цитирования
>> Второй уровень цитирования
>>> Третий уровень цитирования

### Таблици
```sh
| Заголовок 1 | Заголовок 2 |
| ----------- | ----------- |
| Ячейка 1    | Ячейка 2   |
| Ячейка 3    | Ячейка 4   |
```
Пример:
 Заголовок 1 | Заголовок 2 | 
 ----------- | ----------- |
 Ячейка 1    | Ячейка 2    |
  Ячейка 3   | Ячейка 4    |

### Изображение
```sh
![Текст описания](https://www.example.com/image.jpg)
```
Пример

![Щенок](щенок.jpg)
