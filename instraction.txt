Как взять себе репрезиторий:
1. найти нужный на github
2. скопировать ссылку
3. Написать в git: git clone копия ссылки.

Как отправить репрезиторий на github:
1. Создать аккаунт
2. Выбрать: создать новы репрезиторий.
3. в нем выбрать ... or push...
4. скопировать все конды и вставить и в git на компьютере
5. Обязательно подружить ПК с github

Чтобы отправить изменения внесенные на ПК в github, нужно набрать: git push
Чтобы перенести на ПК изменения из github, нужно набрать git pull

Чтобы участвовать в чужом открытом проекте:
1. ищем проект на сайте
2. жмем Fork
3. Копируем ссылку и добавляем ее на ПК чере git clone
4. Делаем отдельную ветку, создаем там нужный файл
5. После всех изменений, отправляем их на github через push
6. жмем на сайте кнопку pull reguest
