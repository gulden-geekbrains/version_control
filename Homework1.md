   # Команды Git

   ## Инициализация нового репозитория.
   ```sh
   git init
   ```
   ## Добавление изменений в индекс
   ```sh 
   git add <файл(ы)>
   ```
   ## Создание коммита с зафиксированными изменениями
   ```sh
   git commit -m "Ваше сообщение коммита"
   ```
   ## Просмотр истории коммитов
   ```sh
   git log
   ```
