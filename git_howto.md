# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Клонировать репозиторий

```sh
git clone <ссылка>
```

## Стянуть репозиторий
```sh
git pull
```

## Залить репозиторий 
```sh
git push
```

## Создать pull request
```sh
git push --set-upstream origin remote
```
