# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Добавление файла для отслеживания

```sh
git add <file_name>
```

## Добавление коммита с комментарием

```sh
git commit -m "Message"
```

## Вызов журнала коммитов

### Вызов развёрнутой версии журнала коммитов

```sh
git log
```

### Вызов однострочной версии журнала коммитов

```sh
git log --oneline
```

### Вызов однострочной версии журнала коммитов с графическим отображением

```sh
git log --oneline --graph
```

## Перемещение по коммитам и веткам

```sh
git checkout <branch_name>
```

## Просмотр веток 

```sh
git branch
```

*Перед активной веткой будет стоять **звёздочка**.*

*Чтобы добавить новую ветку, нужно после команды указать **имя новой ветки**.  Чтобы удалить ветку, перед её именем можно использовать ключ **-d**.*    

```sh 
git branch <new_branch_name>

git branch -d <branch_to_delete_name>
```
## Слияние веток

```sh
git merge
```
*Перед слиянием необходимо перейти в ту ветку, в которую будем вливать информацию*

## Список команд для работы с удалёнными репозиториями

### Клонирование удалённого репозитория на локальное хранилище

```sh
git clone <git_link>
```

### Вытягивание данных из удалённого репозитория

```sh
git pull
```
### Отправка данных на удалённый репозиторий

```sh
git push 
```

### Вливание данных из удалённого репозитория в текущую ветку

```sh
git fetch
```
