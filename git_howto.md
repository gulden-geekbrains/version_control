# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```
Добавляем в commit:
```sh
git add <file>
```

Сделаем commit:
```sh
git commit -m "Message"
```

Отображает все commit:
```sh
git log
```

Отображает все commit кратко:
```sh
git log --oneline
```

Переключение по веткам:
```sh
git checkout <branch_name>
```

Слияние веток
```sh
git merge <name_file>
```

Отображение веток
```sh
git branch
```

Создание ветки
```sh
git branch <name_branch>
```
Клонирование удаленного репозитории
```sh
git clone <url-адрес репозитория>
```

Получение изменение и слияние с локалной версией
```sh
git pull
```

Отправляет локалную версию репозитория на внешний
