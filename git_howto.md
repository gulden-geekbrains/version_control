# Подсказка по гиту

## 1. Инициализация репозитория

```sh
git init
```

## 2. Статус репозитория

```sh
git status
```

## 3. Сохранение файла репозитория

```sh
git add
```

## 4. Комментарий к измененному сохранению
```sh
git commit -m "..."
```

## 5. Журнал изменений репозитория (Список коммитов)
```sh
git log / git log --oneline --graph / git log --graph
```

## 6. Список ветвей репозитория
```sh
git branch
```

## 7. Создать новую ветку
```sh
git checkout -b "имя ветки" / git branch "имя ветки"
```

## 8. Перейти на другую ветку / перейти на другое обновление ветки
```sh
git checkout "имя ветки" / git checkout "адрес нужного обновления"
```

## 9. Удалить ветку
```sh
git branch -d "имя ветки"
```

## 10. Изменить имя главной ветки
```sh
git branch -M "имя ветки"
```

## 11. Слияние веток (Чтобы это сделать, нужно перейти в ту ветку, куда хотите залить всю информации и залить информацию из необходимой:)
```sh
git checkout master -> git merge vetka_2
```

## 12. Копирование удаленного репозитория (Для GitHub необходимо зайти в нужный репозиторий, нажать Code и скопировать https репозитория, затем ввести команду:)
```sh
git clone "скопированная ссылка репозитория"
```

## 13. Отправление своего репозитория на GitHub
```sh
git push
```

## 14. Слияние удаленного репозитория с локальным
```sh
git pull
```
