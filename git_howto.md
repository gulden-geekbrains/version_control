# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```
## Создание копии репозитория: 
``````sh
 git clone git@github.com:*YOURE_GITHUB*/*REPOSITORY_NAME_*.git 
``````
## Отправка изменений: 
``````sh 
git push 
`````` 
## "Взятие" изменений: 
``````sh
 git pull 
 ``````
