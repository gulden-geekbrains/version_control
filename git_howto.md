# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```
## Просмотр веток 
```sh
git branch
```
## Переход между ветками
```sh
git checkout [Имя ветки]
```
## Объединение веток 
```sh
git merge [Имя ветки]
```
## Удаление ветки 
```sh
git branch -d [Имя ветки]
```
## Создание новой ветки
```sh
git branch [Имя ветки]
```
## Показать график изменений
```sh
git log --graph --oneline
```
