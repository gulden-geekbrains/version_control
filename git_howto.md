# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Добавление версии

```sh
git add <file name> 
```

## Добавление коммита

```sh
git commit -m “<message>”
```

## Add и commit одной командой

```sh
git commit -am “<message>” 
```

## Отображение всех коммитов

```sh
git log
```
--oneline *Более компактное отображение*
--graph *Отображение структуры веток*
