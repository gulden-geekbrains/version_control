# Подсказка по git

## Инициализация репозитория

```sh
git init
```

main commands

## Статус репозитория

```sh
git status
```
## Добавить файл к отслеживаю в git

```sh
git add filename.txt
```
## Сохранить изменения с комментарием "..."

```sh
git commit -m "comment"
```

##  Список всех коммитов

```sh
git log
```
