# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

# Подсказки по командной строке 

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. (Для MacOs, Linux)
```sh
pwd
```

Листинг для текущей директории
Windos:
```sh
dir
```
MacOs, Linux:
```sh
ls
```

Удаление файла в Windos:
```sh
del <filename>
```
MacOs, Linux:
```sh
rm <filename>
```
