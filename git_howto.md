# Подсказки по GIT

### Инициализация ГИТа
```sh
git init
```

### Добавление файла к отслеживанию
```sh
git add
```

### Коммит файла
```sh
git commit -m 
```

### Отображение всех коммитов
```sh
git log
```

### Отображение всех коммитов в одну строку
```sh
git log --oneline
```

### Переключение на нужный коммит
```sh
git checkout
```

### Отображение всех веток
```sh
git branch
```

### Перемещение на ветку
```sh
git checkout <название ветки>
```
