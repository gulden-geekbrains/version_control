# Подсказка по гиту

## Инициализация репозитория

```sh
git init
```

## Статус репозитория

```sh
git status
```

## Слияние веток

```sh
git merge
```

## Клонирование внешнего репозитория на локальный ПК

```sh
git clone
```

## Получение изменений и слияние с локальной версией

```sh
git pull
```
## Отправить локальную версию репозитория на внешний репозиторий

```sh
git push

```
