# Подсказка по GIT

## Инициализация репозитория:

```sh
git init
```

## Статус репозитория:

```sh
git status
```

## Добавить файлы к следующему коммиту:

```sh
git add
```

## Переход от одного коммита к другому:

```sh
git checkout
```

## Увидеть разницу между текущим файлом и закоммиченным файлом:

```sh
git diff
```
