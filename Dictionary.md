# Словарь командной строки

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории для мак ос и линукс
```sh
pwd
```

Листинг текущей дирректории
Windows:
```sh
dir
```
Linux MacOs 
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
в linux, macos:
```sh
rm <filename>
