# Подсказки по cmd

Команда смены дирекотрии
````sh
cd c:folder_name
cd "c:folder_name"
````

Команда отображения текущей директорийю MocOS, Linux
````sh
pwd
````

Листинг текущей директорий
Windows
````sh
dir
````
Linux, MacOS:
````
ls
````

удаление файла в Windows:
````sh
del <filename>
````
в Linux, MacOS
````sh
rm <filename>
````

Очистить командную строку
Linux, MacOS
````sh
clear
````
Windows
````sh
cls
````
