# Подсказки по командной строке

Команда смены директории 
```sh
cd c:/folder_name
```

Команда отоброажения текущей директории.MacOs, Linux
```sh
pwd
```

Листинг текущеё директории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файла в Windows:
```sh
del <Filename>
```

В Linux, MacOs:
```sh
rm
```
