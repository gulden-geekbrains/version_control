# Подсказки по командной строке 

Команда смены директории
```sh
cd c:\folder_name
```
\
Команда отображения текущей директории на операционной системе MacOs
```sh
pvd
```
\
Листинг текущей директории на Windows:
```sh
dir
```
на MacOs, Linux:
```sh
ls
```
\
Удаление файла из директории Windows:
```sh
del <filename>
```
MacOs, Linux:
```sh
rm <filename>
```
