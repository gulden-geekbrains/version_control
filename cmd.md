# Подсказка по командной строке

Команда смены директории 
```sh
cd c:\users\Владимир\Desktop\lesson01
```

Вернуться в прошлый репозиторий
```sh
cd ..
```

Листинг текущей директории Windows: 
```sh
dir
```

Удаление файла в Windows:
```sh
del <Название файла>
```

Создать новый файл
```sh
mkdir "Название файла"
```
