# Подсказки по командной строке

Команда смены директории
```sh
cd C:\folder_name
```

Команда отображения текущей директории (для MacOs, Linux)
```sh
pwd
```

Листинг текущей директории
```sh
dir
```
для MacOs, Linux:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
для MacOs, Linux:
```sh
rm <filename>
```
