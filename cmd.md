# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. MacOc, Linux
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, MacOc:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
В Linux, MacOc
``` sh
rm <filename>
```
