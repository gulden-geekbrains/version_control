# Подсказки по командной строке

Команда смены директории
```sh
cd C:\folder_name
```

Команда отображения текуще директории (Linux и Mac)
```sh
pwd
```

Листинг (вывод на экран списка файлов и каталогов, находящихся в даной директории) текущей директории (Windows)
```sh
dir
```

(Linux и Mac)
```sh
ls
```

Удаление файла (Windows)
```sh
del <faile_name>
```

(Linux и Mac)
```sh
rm <faile_name>
```
