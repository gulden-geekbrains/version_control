# Git

###  Создание репозитория
```sh
git init
```

### Добавить к индексации
```sh
git add
```

### Фиксация изменения
```sh
git commit -m ""
```

### Логи/сокращено
```sh
git log
```

### Перемещение по веткам
```sh
git checkout (имя ветки)
```

### Просмотр всех веток
```sh
git branch
```

### Слияние веток
```sh
git merge
```

### Удаление ветки
```sh
branch -d
```

### graph
```sh
log --graph
```

### Создание новой ветки и переключение на неё
```sh
git cheackout -b <branch name>
```

### Загрузка и выгрузка репозитория GitHub
```sh
git pull
git push
```
