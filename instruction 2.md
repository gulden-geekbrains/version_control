# Подсказки по GIT

### Создание репозитория
```sh
git init
```
### Добавление файлов для отслеживания
```sh
git add
```
### Фиксация произведенных изменений в индексе
```sh
git commit -m "New message"
```
### Просмотр зафиксированых изменений (коммитов) в обратном порядке
```sh
git log
```
### Просмотр первых строк зафиксированных изменений (коммитов) в обратном порядке
```sh
git log --oneline
```
### Просмотр строк зафиксированных изменений (коммитов) в виде деревьев
```sh
git log --oneline --graph
```
### Перемещение между зафиксированными изменениями (коммитами)
```sh
git checkout <commit>
```
### Перемещение между ветками 
```sh
git checkout <name_branch>
```
### Просмотр веток 
```sh
git branch
```
### Добавление веток
```sh
git branch <name_branch>
```
### Удаление веток
```sh
git branch -d <name_branch>
```
