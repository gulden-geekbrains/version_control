# Подсказка по GIT

##  Создание репозитория

### Домашнее задание

```sh
git init
```
```sh
git add
```
```sh
git commit -m "сообщение"
```
```sh
git log
git log --oneline
git log --oneline --graph

```
```sh
git checkout "название ветки"
```

```sh
git status
```

```sh
git branch "название новой ветки"
git branch -d "удаление ветки"
```

```sh
git merge "название ветки которую хотим добавить"
```
