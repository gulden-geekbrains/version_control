# Инструкция к языку MarkDown

## Стилизация текста
### Полужирный и курсивный шрифт

Чтобы задать для текста полужирный шрифт, заключите его в двойные звездочки или двойное нижнее подчеркивание:

**ПОЛУЖИРНЫЙ ШРИФТ** или __ТАКОЙ ПОЛУЖИРНЫЙ ШРИФТ__

Чтобы задать для текста курсивный шрифт, заключите его в одинарные звездочки :

*КУРСИВНЫЙ ШРИФТ* или _ТАКОЙ КУРСИВНЫЙ ШРИФТ_

__*Чтобы одновременно использовать курсивный и полужирный необходимо использовать сочетание "**_ _**", или "*__ __*" *__

### Заголовки

Для создания заголовка используется символ *#*, адля создания подзаголовка необходимо использовать *##*. Далее каджый последующий подзаголовок необходимо увеличивать количество данного символа. Например:
```sh
# Заголовок статьи 
## Подзаголовок статьи
### Подзаголовок подзаголовка статьи 
и т.д. 
```
### Блок цитирования


Блоки цитирования создаются с помощью символа >.И при каждом уровне цитирования приводит к увеличению количества символа >>
>первый уровень цитирования
>>Второй уровень цитирования

### Списки
Для создания ненумерованного списка используйте - или *, за которым следует пробел в начале каждой строки:
* пункт1
* пункт2
* пункт3

Нумерованный список
1. пункт1
2. пункт2
3. пункт3
