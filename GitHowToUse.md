# Инструкция по работе с Git

## Инициализация локального репозитория
``` sh
git init
```
## Получение информации о текущем состоянии
``` sh
git status
```
## Добавление файла к следующему коммиту
``` sh
git add
```
## Создание коммита
``` sh
git commit -m "message"
```
## Вывод на экран истории всех коммитов с их хэш-кодами
``` sh
git log
```
## Переход от одного коммита к другому
``` sh
git checkout
```
## Возврат к актуальному состоянию и продолжение работы
``` sh
git checkout master
```
## Разница между текущим и закоммиченным файлами
``` sh
git diff
```
## Посмотреть список веток в репозитории
``` sh
git branch
```
## Создать новую ветку
``` sh
git branch <name>
```
## Удалить ветку
``` sh
git branch -d <name>
```
## Перейти между ветками
``` sh
git checkout <name>
```
## Слить ветки
``` sh
git merge
```
